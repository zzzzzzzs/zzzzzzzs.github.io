                        # 配置文件

                        修改配置文件，主要是修改数据库项目的东西

                        cd zlink-release-{version}/config
vim application.yml

                        # 数据库初始化

                        首先创建 zlink 数据库

                        CREATE DATABASE `zlink` CHARACTER SET 'utf8mb4' COLLATE 'utf8mb4_general_ci'

                        然后在数据库中执行 zlink-release-{version}/config/db/zlink.sql 文件中的内容
